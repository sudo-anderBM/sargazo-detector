De las 13 bandas disponibles, el algoritmo AFAI necesita exactamente tres más la máscara de nubes:

B04 Rojo — absorción del sargazo 665 nm
B8A NIR — alta reflectancia en algas 865 nm
B11 SWIR — distingue agua vs vegetación 1610 nm
SCL Máscara de nubes 20 m
.SAFE / GRANULE / L2A_* / IMG_DATA /

03 — Algoritmo Principal

Índice AFAI

Wang & Hu, 2009 — Remote Sensing of Environment
AFAI = B8A − B8A*
B8A* = B04 + (B11 − B04) × (865 − 665) / (1610 − 665)
B8A* = NIR esperado si sólo hubiera agua (interpolación lineal rojo→SWIR)
01
Normalización
Valores crudos ÷ 10,000 → reflectancia [0,1]. Píxeles ≤0 marcados como NaN.
02
Cálculo AFAI
Operación píxel a píxel sobre arrays NumPy. AFAI positivo = posible sargazo.
03
Umbral Otsu
Análisis estadístico del histograma. Umbral óptimo calculado automáticamente.
04 — Innovación Metodológica

Umbral Adaptativo de Otsu

AFAI bajo (agua) ↑ Umbral 0.1307 AFAI alto (sargazo)
✗ Umbral Fijo (ingenuo)
AFAI > 0.005 = sargazo

Falla en distintas imágenes. El valor óptimo cambia con la iluminación, estación del año y condiciones atmosféricas.
✓ Otsu (implementado)
threshold_otsu(afai)

Maximiza la varianza inter-clase estadísticamente. Funciona de forma óptima en cualquier imagen sin ajuste manual.

06 — Implementación

Arquitectura del Código

def calcular_afai(B4, B8A, B11):
  # Longitudes de onda Sentinel-2 (nm)
  λ_red, λ_nir, λ_swir = 665, 865, 1610
  B8A_prima = B4 + (B11 - B4) * ((λ_nir - λ_red) / (λ_swir - λ_red))
  return B8A - B8A_prima

def detectar_sargazo(afai):
  # Umbral adaptativo — no fijo
  datos = afai[~np.isnan(afai)]
  umbral = threshold_otsu(datos)
  return afai > umbral, umbral
